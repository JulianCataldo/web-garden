Example usage with SCSS:

```scss

  // â€¦navbar stylesâ€¦

  transition: all 1s;

    opacity: 1;
    transform: translateY(0);
    &::after {
      content: 'Is scrolling upâ€¦';
    }
  }
  [data-is-top='true'] & {
    opacity: 1;
    transform: translateY(0);
    &::after {
      content: 'Is top';
    }
  }
  [data-is-bottom='true'] & {
    opacity: 1;
    transform: translateY(0);
    &::after {
      content: 'Is bottom';
    }
  }
}
```
